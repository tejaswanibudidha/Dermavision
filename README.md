# DermaVision AI

A modern Skin Disease Detection website UI built with React and Tailwind CSS.

## Prerequisites

- [Node.js](https://nodejs.org/) (Latest LTS version)

## Setup and Installation

1.  **Install Dependencies**
    Open your terminal in this directory and run:
    ```bash
    npm install
    ```

2.  **Run Development Server**
    Start the local development server:
    ```bash
    npm run dev
    ```

3.  **Build for Production**
    To create a production build:
    ```bash
    npm run build
    ```

## Project Structure

- `src/components`: Reusable UI components (Navbar, Sidebar, etc.)
- `src/layouts`: Page layouts (MainLayout, DashboardLayout)
- `src/pages`: Individual pages (Home, Login, Dashboard, etc.)
- `src/assets`: Static assets

## Features

- **Public Pages**: Home, Login, Register, About, Contact
- **User Dashboard**: Profile management
- **Analysis Workflow**: Image upload with drag-and-drop
- **Results Visualization**:
    - Disease identification
    - Confidence score
    - Precautions and care
    - Dietary recommendations
